# OrbitX + MAITRI AI

**Space Exploration Platform + Emotion-Aware Wellbeing Companion**

Built by Team CypherItX for Ingenious — Viva Institute of Technology

## Project Structure
```
ORBITX/
├── assets/          → Videos, images, logos
├── auth/            → Login (HTML + PHP)
├── dashboard/       → Dashboard (HTML + CSS + JS)
├── explore/         → Explore section
├── home/            → Home page
├── signup/          → Signup (HTML + PHP)
├── MAITRI_AI/
│   ├── Pys/         → Python AI modules
│   ├── app.py       → Flask backend
│   └── MAITRI_AI.html
├── landingpge.html  → Landing page
└── test.php         → DB connection test
```

## Setup

### OrbitX (PHP + MySQL)
- Import `database/orbitx_db.sql` into MySQL
- Update DB credentials in `auth/login.php` and `signup/signup.php`
- Run via XAMPP

### MAITRI AI (Python)
```bash
cd MAITRI_AI/Pys
pip install -r requirements.txt
python app.py
```
